## What This Is

This is simply a collection of my Python pet projects.  
Nothing supernatural here, but each file is a little story about how I tried, experimented, or just killed time with some benefit.

Some things actually work, some are just a demonstration of an idea.  
But all of them are part of my journey.

## What's Inside

1. **Slot Machine** — almost like in a casino, but without money. Animation, jackpots, statistics, and a bit of excitement in the console.
2. **Translator** — back and forth between Russian and Chinese. Voice, clipboard, batch processing. Handy when you're too lazy to Google.
3. **30 Tasks** — from simple to intermediate. Just to keep the brain from rusting.
4. **Geometry Library** — circles, triangles, polygons. Computes everything, validates everything.
5. **System Monitor** — looks at CPU, memory, disk, video. Useful when you think your computer is slowing down.
6. **Assistant** — a set of commands for work, fun, and various small things. Extendable if you want.
7. **And much more** — look in the folders, there's something to see.

## Technical Details

- **Language:** Python 3.8+
- **External libraries:** minimal. Some use `requests`, `psutil`, `gTTS`, `pydub`, `googletrans` — all install with one command.
- **Graphics:** where needed — Tkinter.
- **Main rule:** works everywhere Python works.

## How to Run

```bash
git clone https://github.com/just-not-google/all-my-little-works.git
cd all-my-little-works
python filename.py
```

Each script is standalone. Open and run.

1. **Слот-машина** — почти как в казино, но без денег. Анимация, джекпоты, статистика и немного азарта в консоли.
